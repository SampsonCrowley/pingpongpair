# Ruby Dev Setup

## Installation


clone this repo into your project folder

```git
git clone https://github.com/SampsonCrowley/ruby_dev_setup.git
```

cd into the ruby_dev_setup dir:

    $ cd ruby_dev_setup

run ./setup:

    $ ./setup

delete the ruby_dev_setup dir. you don't need it now

    $ cd ..
    $ rm -r ruby_dev_setup
